S4 Scene Editor - Angular and ThreeJS demo

Requires desktop browser -- Chrome, FireFox, or IE11

This demo shows:

  1. AngularJS binding for commands and data input
  2. AngularJS async ajax-like JSON resource retrieval
  3. JSON parsing (Using S4 -- "Steve's Super Simple Scene Format")
  4. Three.JS animation
  5. Live scene editing
