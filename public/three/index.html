S4F Scene Editor

Requires desktop browser -- Chrome, FireFox, or IE11 (IE has issues with range control --WIP)

Demo features

  1. AngularJS binding for commands and data input
  2. AngularJS async ajax-like JSON resource retrieval
  3. JSON parsing (Using S4F -- "Steve's Super Simple Scene Format")
  4. Three.JS animation
  5. Live scene editing
